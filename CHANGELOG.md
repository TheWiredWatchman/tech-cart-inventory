# Changelog

All notable changes to the Tech Cart Inventory System will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-14

### Added
- Initial public release
- Multi-shift cart inventory tracking
- Cable inventory with slot positions
- Laptop specifications display
- Software inventory with version tracking
- Real-time search across all categories
- Stats dashboard (cables, software count, storage, RAM)
- Responsive design for mobile devices
- Konami code easter egg (Matrix effect)
- Fat Maintenance Man mini-game easter egg
- High score system with local storage

### Features
- Single HTML file architecture (no dependencies)
- Works offline
- Easy customization through JavaScript object
- Glassmorphism UI design
- Smooth animations and transitions

### Industrial Software Support
- KEYENCE (KV Studio, IV-Navigator, etc.)
- Omron (CX-One, Sysmac Studio)
- Mitsubishi (GX Works, GT Designer, RT ToolBox)
- Basler/Sentech camera software
- Various HMI editors
